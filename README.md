# Computacao-Grafica-com-OpenGL-e-OpenAL
Projeto final da disciplina de computação gráfica da UTFPR, criado utilizando a API OpenGL para renderização, OpenAL para audio e a biblioteca SOIL para aplicação de texturas.

Controles:
    WASD -> Movimentação;
    Space -> Pulo;
    B -> Backflip;
    X -> execução do audio.
